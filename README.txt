LIBRERÍA TEORÍA CUANTICA, OBSERVABLES Y MEDIDAS

En esta librería se presenta el programa para realizar operaciones básicas con números complejos y con ellas lograr hacer diversas operaciones con matrices y vectores.

Pre-requisitos.
Es necesario descargar el lenguaje de programación Python.
Obtener una copia de la librería publicada en el repositorio.

Descargar e Instalar Python.
1. Ir al siguiente link: https://www.python.org/downloads/
2. Hacer clic en el botón “Download Python”
3. Una vez completada la descarga, abra el instalador, presione “siguiente” y aceptar los términos y condiciones de la aplicación.

Instalación.	
Obtener una copia del Proyecto.
1. Dijirse al link del repositorio: git remote add origin https://github.com/JoseH26/Teoria-cuantica-medidas-y-observables.git

2. Pulsar el botón “Code” de color verde.
3. Seleccionar la opción “Download ZIP”
4. Descomprimir el .ZIP y ejecutar el archivo .py

Ejecutando las pruebas.
Para realizar pruebas en la librería, abrir python y dar click en "File", luego "open" y seleccionar el archivo “Medidas y Observables JH.py”, una vez allí encontrará todas las operaciones principales de los numeros complejos,
así mismo algunas funciones en el entorno de los vectores y las matrices que se complementan entre si, una vez demos click al boton de ejecutar al programa podemos desarrollar los ejercicios propuestos con datos propios del usuario, este proceso se puede realizar cuantas veces lo deseemos.
 
Construido con
Python - IDLE (Python 3.9 64-bit)
